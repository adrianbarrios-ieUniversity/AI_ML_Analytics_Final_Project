Recording Details:
Each experiment was recorded and split into the individual files attached here. All data was converted to single channel (taking the left channel if the recording was stereo) at a 44,100Hz sampling rate.

Background
The Gunshot Audio Forensics Dataset is a set of individual data files collected as part of NIJ
Grant 2016-DN-BX-0183. Data was collected in Summer 2017 in rural Arizona. Each of twenty
firearms were recorded from each of twenty different positions on four different recording
devices. The dataset contains approximately 10,000 individual gunshot recordings. The project
team included:
• Ryan Lilien (Cadre Research; Chicago, IL)
• Jason Housman (Tulane University; New Orleans, LA)
• Ram Mettu (Tulane University; New Orleans, LA)
• Todd Weller (Weller Forensics; Burlingame, CA)
• Lucien Haag (Forensic Science Services; Carefree, AZ)
• Michael Haag (Forensic Science Consultants; Albuquerque, NM)
Geometries
Each firearm was recorded from twenty different recording positions (yellow star, purple circle,
green triangle). The shooter was positioned at the red triangle and shot towards the indicated
target.
Files
Each zip file contains multiple gunshot recordings from a range of recording positions for a
specific combination of a recording device and firearm.
The files follow the naming convention XX_YYYY_SZZ.wav (see below for details)
XX: Device Prefix
YYYY: Experiment ID (Recording Geometry)
ZZ: Shot number
Device Prefixes:
ZM: Zoom H4N
IP: iPhone 7
SA: Samsung Galaxy S7
Experiment ID (Recording Geometry):
The experiment ID has three numbers and then a letter. The geometries of the experiments are
related to the numbering of the experiment ID. The following table lists the geometries
corresponding to the number and letter Experiment ID. Taken together ZM_041A_S02.wav
would correspond to the Zoom recording device positioned at 90 degrees and 40 meters from
the shooter and this file would contain shot 2. See diagram above for recording geometries.
